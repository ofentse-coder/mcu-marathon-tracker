# MCU Marathon Tracker

A web application to help new MCU fans catch up on all the movies and series in the Marvel Cinematic Universe.

## 📖 About

Whether you're brand new to the MCU or looking to organize your watch journey, MCU Marathon Tracker is here to help you navigate the vast universe of Marvel films and TV series. This tool provides a structured way to track your progress through the interconnected stories of the MCU.

## ✨ Features

- Track movies and series across the MCU timeline
- Keep notes on your viewing progress
- Organize watchlist by release order or chronological order
- User-friendly interface for planning your MCU marathon

## 🛠️ Technology Stack

This project is built with:
- **HTML** (38.2%) - Structure and markup
- **JavaScript** (37.8%) - Interactivity and functionality
- **CSS** (24%) - Styling and layout

## 📦 Installation

1. Clone this repository:
```bash
git clone https://github.com/ofentse-coder/mcu-marathon-tracker.git
```

2. Navigate to the project directory:
```bash
cd mcu-marathon-tracker
```

3. Open `index.html` in your web browser or serve the files using a local server

## 🚀 Usage

1. Open the application in your web browser
2. Browse through the MCU movies and series
3. Mark content as watched or add to your watchlist
4. Keep track of your MCU marathon progress

## 🎬 MCU Content

The tracker includes content from:
- MCU Films (Phases 1-6)
- MCU TV Series (Disney+, streaming, and classic series)
- Supporting narratives and spin-offs
